# TASK 2 - DEEP LEARNING PROJECT

## Objective

Build a Deep Learning model for handwritten digit recognition using TensorFlow and the MNIST dataset.

## Tools Used

* Python
* TensorFlow
* Keras

## Dataset

MNIST Handwritten Digits Dataset

## Model Architecture

* Input Layer (28x28 Images)
* Flatten Layer
* Dense Layer (128 neurons, ReLU activation)
* Output Layer (10 neurons, Softmax activation)

## Steps Performed

1. Loaded the MNIST dataset.
2. Normalized image pixel values.
3. Built a neural network using TensorFlow/Keras.
4. Trained the model for 5 epochs.
5. Evaluated the model on test data.
6. Saved the trained model.

## Result

The model achieved a test accuracy of approximately 97.74% in recognizing handwritten digits.
